# AI Issue Tracking Template

This template is designed for AI-driven issue tracking. 

**Instructions for AI:**
1. **READ** the `spec/` directory thoroughly to understand the system architecture, workflows, and data structures.
2. **FIND** any logical errors, inconsistencies, or unhandled edge cases within the specifications or current implementation.
3. **REPORT** any discovered issues by creating a new issue document using the format below.
